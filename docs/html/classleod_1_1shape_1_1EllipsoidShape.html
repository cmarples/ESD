Detailed Description

The ellipsoid shape class.

Contains the principal semi-axis lengths; \(a\), \(b\) and \(c\) of an ellipsoid. Also includes functions for conversion between different coordinate systems on the ellipsoid surface.

◆ normalise()

def normalise (   self)

Scale axes so that the effective radius \(abc^{1/3}\) equals unity, or equivalently so the volume equals \(4\pi/3\) (i.e.

that of a unit sphere). This is useful for studying how some quantity varies with aspect ratio, while keeping the size (volume) constant.
