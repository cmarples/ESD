Calculates the distance between given start and end points on an ellipsoid, with a given shape and mesh.

Parameters
shape: EllipsoidShape
The ellipsoid.
mesh: FmmMesh
The mesh of vertices defined on the ellipsoid surface (can be polar or icosahedral).
start_point: list of floats (2 elements)
The start point, in \((\theta, \phi)\) coordinates.
end_point: list of list of floats ( \(n\) elements)
Set of \(n\) end points, in \((\theta, \phi)\) coordinates.
is_dijkstra: bool (optional)
If True, Dijkstra's algorithm is used. Otherwise, the fast marching method is used. Defaults to False.
is_radians: bool (optional)
Specifies whether the elements start_point and end_point are given in radians (True) or degrees (False). If False, a coordiante conversion to radians is performed. Defaults to False.
Returns
[list of floats, FmmResult]
The first element is a list of obtained distances from the start_point to all points in end_point. The second element contains more detailed information on the performed calculation.
